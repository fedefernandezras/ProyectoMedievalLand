Se realizaron las siguientes prácticas y consideraciones;
 -optimización de imágenes:
 Se redujeron de tamaño y peso los archivos .jpg y .png, para optimizar su rendimiento. Para ello utilicé diversas herramientas, como la extensión de vsc tinypng-compress, también las web squoosh.app y rezisepixel.
-keywords:
 Se añadieron keywords a cada archivo .html, para facilitar su acceso. Se tuvo en cuenta las palabras más usadas por los usuarios para buscar este tipo de contenido.
-Títulos:
se añadieron títulos diferentes a cada .html, para facilitar su búsqueda y para mejorar el entendimiento de usuario.
-Descripciones:
se añadieron descripciones llamativas para cada .html, utilizando la metaetiqueta "description".
-Uso estructurado de H1, H2 etc.
-Moderada cantidad de encabezados por página.
